2 Hexo主题

今天，在知乎上看到一个帖子，有哪些好看的 Hexo 主题？，准备给自己的blog找一个好看的主题。然后看到某位知友minhow的主题之后甚是喜欢，因此决定依葫芦画瓢，自己也捣鼓一下。该知友的个人blog。

2.1 初始化本地博客

输入创建命令创建并初始化本地博客：

$ hexo init username.github.io // 建议和创建仓库时使用同一个，我的为allbluelai.github.io

因为我已经安装过滤，这里就不再回车了。
